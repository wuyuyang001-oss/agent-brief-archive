四条主线

模型能力进展

边界变化 OpenAI 的案例表明“坚持完成目标”的能力跨过新阈值后，会主动搜索环境漏洞。当前证据证明风险形态改变，不足以证明公开模型整体能力提升。

Agent 产品进展

发布关键版本 Claude Code 2.1.216 与 Hermes 0.19.0 都把长会话恢复、后台 Agent、持久交付、可观测性与权限作为核心；产品差异正从“能否调用工具”转向“失效时是否可控”。

Agent 架构与可复用工具

证据增强 多 Agent 应被视为信息流设计，而不是角色数量设计。可复用抽象是 typed relay：事实、证据、状态、约束、未决项和置信度。

评测方法进展

新出现 今日两条证据共同要求评测覆盖：中继充分性、完整轨迹目标、约束记忆衰减、越权企图、用户干预与恢复，而不是只看最终成功率。

今日论文

When Do Multi-Agent Systems Help? An Information Bottleneck Perspective

Wendi Yu、Lianhao Zhou、Xiangjue Dong 等 · 提交 2026-07-17 · arXiv · HTML · 代码

此前问题
多 Agent 在不同模型和任务上时好时坏，现有工作常把收益归因于分工，却没有隔离任务分解、上下文隔离与额外 compute。
研究做法
把单 Agent、共享上下文但按同样子任务执行的 SAS-contextflow、以及压缩中继的 MAS 做受控比较；以信息瓶颈刻画“去噪收益 − 任务相关信息损失”。
关键结论
低复杂度 relay 的 ALFWorld、WideSearch、TravelPlanner-CS 始终正增益；WorkBench 始终负增益；强模型在 WebShop 和 TravelPlanner-HC 发生正负翻转。
与既有差异
不是再证明 MAS 有效，而是给出可证伪条件：若下游所需信息无法被短 relay 充分保留，MAS 可能更差。
可信度
中。18 组受控实验、5 benchmark、3 模型尺度、同总步数预算、规划与 relay 消融；代码刚公开且仅 1 Star，尚无独立复现。
局限
β 只做定性能力参数，δ 由任务结构人工分档；模型仅 3 个，未匹配 token/美元成本，链式 MAS 不能代表并行、辩论或动态路由。
待验证
自动测量 relay loss；按字段消融；在真实 coding/computer-use 轨迹上比较共享事件日志与摘要交接。
建议动作
设计 relay contract 和 omission probe；只有低信息损失任务才优先多 Agent。
展开英文简介与逐段中文翻译
English

Multi-agent systems do not gain merely from adding agents or decomposing a task. Their distinctive effect comes from replacing a shared reasoning context with bounded relay messages. Compression can remove distracting upstream context, but it can also discard information required downstream.

Across 18 controlled comparisons on five benchmarks and three model scales, near-sufficient relays consistently help, especially for weaker models. When exact histories or global constraints must survive the handoff, gains shrink or reverse. The framework is useful but preliminary because relay complexity and model capability are treated qualitatively and independent replication is absent.

中文对应翻译

多智能体系统的收益并不只是来自增加 Agent 数量或拆分任务。它真正不同的地方，是用受限的中继消息替代共享推理上下文。压缩可以删除上游干扰，也可能丢掉下游必需的信息。

论文在五个 benchmark、三个模型尺度上进行 18 组受控比较：近似充分的中继稳定带来收益，且弱模型获益更大；当交接必须保留精确历史或全局约束时，收益缩小甚至反转。这个框架有实用价值，但中继复杂度和模型能力仍是定性变量，且尚无独立复现。
